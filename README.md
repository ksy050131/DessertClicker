# 🍬 DessertClicker

디저트를 클릭해서 점수를 모으는 Unity 클리커 게임

## 게임 소개

귀여운 도트 젤리를 클릭하여 **Sweet Points**를 모으고, 상점에서 업그레이드를 구매하세요!

## 주요 기능

- 🖱️ **클릭 시스템** — 젤리를 클릭하면 점수 획득 + 펀치 애니메이션
- 🏪 **업그레이드 상점** — 클릭당 점수 증가, 자동 생산 등 업그레이드 구매
- 🎨 **도트 아트 UI** — Galmuri11 픽셀 폰트 + 9-Slice 도트 패널

## 기술 스택

- **엔진**: Unity (New Input System)
- **언어**: C#
- **해상도**: 1920×1080 (16:9 웹)
- **폰트**: Galmuri11 (TMP Raster, Dynamic 한글 지원)

## 실행 방법

1. Unity Hub에서 프로젝트 열기
2. `Assets/Scenes/SampleScene.unity` 로드
3. ▶️ Play
